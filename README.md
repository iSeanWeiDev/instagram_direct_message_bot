# Instagram Direct Message Bot.

1. How to excute this application?
    1) follow this steps at development method.
        - install the node modules.
            `````````
            npm install
            `````````

        - migrate database.
            `````````
            sequelize db:migrate
            `````````

        - start your application with standard method.

            ````````
            npm start
            ````````
